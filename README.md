# MicroIoT

MicroIoT是基于微服务架构的物联网平台，通过软件定义物联网元数据信息，对物联网进行配置和管理，利用这种集中式的物联网管理方法，将物联网基础设施和应用分离，使得MicroIoT物联网平台可以监视，控制任意类型的“物”，物联网应用与“物”之间、或者“物”与“物”之间的通信可以通过MicroIoT提供的API实现，加快物联网产品上市速度。
